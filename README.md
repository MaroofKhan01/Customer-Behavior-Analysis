# 👨🏻‍💻 Customer Behavior Analysis

Transforming raw customer data into meaningful business insights through an end-to-end **Data Analytics** workflow. This project showcases how data can be used to understand customer behavior, uncover purchasing trends, and support smarter business decisions.

## 🚀 Tech Stack

* 🐍 **Python** – Data Cleaning, Preprocessing & Exploratory Data Analysis (EDA)
* 🗄️ **SQL** – Business Queries & Customer Segmentation
* 📊 **Power BI** – Interactive Dashboards & Data Visualization

## ✨ Key Highlights

* 🔹 Cleaned and transformed raw customer data for accurate analysis.
* 🔹 Explored purchasing patterns and customer behavior using Python.
* 🔹 Solved real-world business questions with SQL queries.
* 🔹 Built an interactive Power BI dashboard to visualize KPIs and trends.
* 🔹 Delivered actionable insights to support data-driven decision-making.

> **📈 A complete portfolio project that demonstrates the end-to-end data analytics lifecycle—from raw data to business insights—using industry-standard tools and best practices.**
